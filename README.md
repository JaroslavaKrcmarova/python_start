# python_start
# Python a Jupyter Notebook Projekty

Toto repository obsahuje mé kódy v Pythonu a Jupyter Notebooky, které jsem si vytvořila během kurzů:

- **Kurz od Michala Hučka** – Pythonu na youtube
- **Kurz "Datová analýza v Pythonu" od Czechitas** – pokročilejší práce s daty, analýza datasetů a praktické úlohy.

## Obsah repository

- `notebooky/` – Jupyter Notebooky z obou kurzů.
- `python_skripty/` – samostatné Python skripty.
- `README.md` – tento soubor s informacemi o repozitáři.

## Požadavky

Pro spuštění notebooků doporučuji mít nainstalováno:

- Python 3.10+  
- Jupyter Notebook nebo JupyterLab  
- Knihovny používané v kurzech: `pandas`, `numpy`, `matplotlib`, `seaborn` (a další dle konkrétních notebooků)

Instalaci můžete provést například přes pip:

```bash
pip install pandas numpy matplotlib seaborn jupyter
